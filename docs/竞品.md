## 2026-07-07 数据爬取快照

### moonlightpeaks.wiki.gg

- 使用 MediaWiki `allpages` API 抓取到 443 个页面标题。
- 可用分类线索：
  - 角色/Profile 线索约 26 个：Mina、Evan、Orlock、Samael、Elvira、Saga、Ridge、Brook、Ludo、Luna、Fiona、Sabrina、Noel、Kim、Rei、Tae、Aras、Alina、Dragan、Death、Jada、Persephone、Winston 等。
  - 家族/家族地点线索约 10 个：Ambrosia、Logan、Webb、Hosu、Khazan，以及 Mansion、Graveyard、House、Webb of Wonders 等关联地点。
  - 地点/商店/地标线索约 26 个：Coffee & Coffins、Howling Hammer、The Broken Lamp、Third Eye Threads、Moonlit Pines、Howling Marshes、Misty Shores、Luna Bay、Museum、Silverveil Lake、Yoga Shack 等。
  - 物品/系统/分类线索约 381 个：crops、flowers、trees、tools、workstations、animals、resources、recipes、decorations、achievements、artifacts 等。
- 抽查角色与商店 raw wikitext：很多角色 infobox 有 portrait 文件名、pronouns、birthday、species、lives in、occupation、family；gift 字段多为空，不能直接写硬表。
- 处理策略：提取事实字段、图片文件名可用性和来源 URL；不复制正文，不复制图片。

### Fandom

- 使用 MediaWiki `allpages` API 抓取到 125 个页面标题。
- 可用分类线索覆盖 Achievements、Characters、Families、Animals、Crops、Fish、Recipes、Tools、Locations、Energy、Friendship、Quests、Seasons、Weather、Nokturna、Romance、Vampires、Werewolves、Witches、Mermaids、Seers 等。
- Fandom 单页正文/抽取结果较稀薄，适合作为实体名交叉验证和未来扩展 bucket，不适合作为最终事实来源。

### moonlightpeakswiki.com

- sitemap 抓取到 375 个 URL。
- 站点结构：
  - tools 相关 12 个 URL，包括 Platform Picker、Romance Gift Finder、Romance Match Quiz、Romance Planner、Item Tracker、Farming Profit Planner、Submit Data。
  - characters 相关 119 个 URL。
  - families 相关 10 个 URL。
  - locations 相关 15 个 URL。
  - items 相关 19 个 URL。
- 可借鉴工具：
  - Platform Picker：根据玩家设备/玩法偏好推荐 Steam、Steam Deck、Switch、Switch 2、Android。
  - Item Tracker：本地清单追踪 crops、flowers、tools、resources、animals、recipes 等。
  - Farming Profit Planner：玩家输入 seed cost、sell price、growth days、harvest count、extra cost 后计算 profit/day。
  - Romance 工具：gift finder、match quiz、planner 的方向可借鉴，但当前 gift 数据未稳定，本站优先做 romance tracker + private notes。

## 监控说明

1. `moonlightpeakswiki.com` 已覆盖 release、demo、platforms、characters、romance、Nokturna、tools、FAQ 和多语言入口；本站应通过更清晰的决策型攻略和内部链接提升质量。
2. `wiki.gg` 可作为数据线索，但不要直接复制数据库；gift、fish、crop、Nokturna 表格需要 live-game 验证后再扩展。
3. 当前保留 PS5/Xbox 为 watch，codes 为 ignore/safety intent；除非官方新增平台或兑换系统，不单独建薄页。
4. 本站已将竞品工具功能重建为 `/tools`、`/tools/platform-picker`、`/tools/romance-tracker`、`/tools/item-tracker`、`/tools/farming-profit-calculator`；以本地保存和人工输入为主，避免发布未验证价格/礼物表。
